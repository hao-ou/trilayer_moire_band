## T-TL WSe2 band calculator
This is the repository used to reproduce the band structure and wavefunction distribution calculation of twisted tri-layer WSe2.
The code is based on the paper of Nakatsuji *et al* [paper](https://www.nature.com/articles/s43246-025-00996-9)

Partly based on the discussion with Gemini, even though I'm afraid that without it maybe I can finish it earlier.
